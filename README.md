# Baasandorj CV Website

A personal portfolio website built with Next.js and TypeScript to present my background in computer science, AI, software development, music performance, and technical projects.

## Overview

This website serves as my online CV and portfolio. It brings together my education, experience, projects, recordings, and contact information in a responsive web app.

## Features

- Responsive portfolio website
- Built with Next.js App Router
- TypeScript-based codebase
- Project and experience sections
- Recordings / media section
- Contact information and external links
- Modular component structure for easy updates

## Tech Stack

- Next.js
- TypeScript
- CSS
- React
- Node.js tooling

## Project Structure

- `app/` – app routes and pages
- `components/` – reusable UI components
- `lib/` – content in English and Mongolian
- `public/` – static assets such as images and media
- `styles/` – styling files
